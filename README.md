### Cómo instalar ?

1. Clona el repositorio ``` git clone https://github.com/tomassiNico/ahorcadosApp_ASO_2020.git ```

2. Busca el apk en ```<direccionrepo>/assets/apk```

3. Copia el apk en tu telefono Android

4. Instala y a jugar!
